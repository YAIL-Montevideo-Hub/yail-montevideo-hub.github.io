## Sobre el proyecto

Fauna Local es una plataforma que permite a cualquier persona identificar animales nativos de Uruguay a partir de una foto. El objetivo es construir una base de datos colaborativa de avistamientos que apoye la investigación en biodiversidad y la educación ambiental.

## Objetivos

- Entrenar un modelo de visión computacional con especies nativas de Uruguay
- Crear una herramienta accesible para ciudadanos, educadores e investigadores
- Generar datos georreferenciados útiles para el seguimiento de poblaciones de fauna silvestre

## Colaboradores buscados

El proyecto busca sumar personas con experiencia en:

- Computer vision y clasificación de imágenes
- Biología o ecología (para validación de datos)
- Desarrollo mobile o web
- Diseño UX
